# Logo Files

This directory contains the Cloud Native Provence logos in various formats.

- `logo-primary.svg` - Primary logo in SVG format
- `logo-primary.png` - Primary logo in PNG format (high resolution)
- `logo-white.svg` - White version for dark backgrounds
- `logo-white.png` - White version in PNG format
- `logo-icon.svg` - Icon/symbol only version
- `logo-icon.png` - Icon in PNG format
